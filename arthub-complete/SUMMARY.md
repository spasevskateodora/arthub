# ArtHub — Резиме на функционалностите

## Опис на апликацијата
ArtHub е онлајн галерија и marketplace за оригинални уметнички дела. Администраторот (уметникот) качува свои дела, а регистрираните корисници можат да ги прегледуваат, пребаруваат и нарачуваат.

---

## Кориснички улоги

### Гост (нерегистриран)
- Прегледува галерија и дела
- Чита reviews
- Пребарува и филтрира дела
- Гледа инспирација од Met Museum

### Купувач (регистриран корисник)
- Сè од гост +
- Регистрација и најава (JWT)
- Нарачување оригинални дела
- Остава reviews
- Преглед на свои нарачки

### Admin
- Сè +
- Додавање дела со upload на слика
- Уредување и бришење дела
- Управување со нарачки (Pending → Confirmed → Completed)
- Dashboard со статистики и графикон

---

## Функционалности

### Галерија
- Прикажување на сите дела во grid
- Филтрирање по категорија
- Пребарување по наслов
- Sold Out overlay на продадени дела
- Вистинскиот број на дела и достапни од базата

### Детален приказ
- Слика, наслов, опис, категорија, medium, димензии, година
- Цена во EUR
- Buy копче → checkout
- Reviews секција со можност за додавање
- Admin: Edit и Delete копчиња

### Checkout
- Форм за податоци на купувач (ime, prezime, email, telefon, adresa, grad, zemja)
- Валидација на сите полиња со regex
- По нарачка → дело автоматски се означува Sold Out
- Email потврда (Nodemailer)

### Admin Dashboard
- Статистики: вкупно дела, нарачки, приход, sold out
- Bar chart (Chart.js) — нарачки по месец
- Breakdown по категорија
- Табела со сите дела + Edit копче
- Листа на нарачки со промена на статус

### Надворешен API
- Met Museum API — вистински уметнички дела прикажани во "Inspiration" секција

### Технички
- REST API со Swagger документација на /api/docs
- JWT автентикација
- bcryptjs за хаширање лозинки
- Multer за upload на слики
- Docker за локален development
- MongoDB за база на податоци
